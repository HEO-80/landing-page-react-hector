## 🚀 Instalación y Ejecución
```bash
git clone https://github.com/HEO-80/landing-page-react-hector.git
cd landing-page-react-hector
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.

**Build de producción:**
```bash
npm run build
npm run preview
```
